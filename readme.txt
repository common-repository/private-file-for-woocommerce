Consente di poter inviare ai tuoi clienti file privati, pagine private ed una pagina condivisa, nella sezione my account di woocommerce nell'area clienti

== Descrizione ==

Private File For Woocommerce è un semplice ma utilissimo plugin che permette di poter allegare file in modo totalmente privato direttamente al singolo cliente, che lo potrà scaricare
nella sezione privato nell'account di woocommerce.


= Che cosa permette di fare? =

Il plugin è un fork del cuore principale di [Wp Customer Area](https://it.wordpress.org/plugins/customer-area/) versione 1.6 -THANKS E CREDIT - assemblato nelle funzionalità di woocommerce.

Oltre a file privati, permette anche di inserire pagine private per singolo cliente, ed infine permette di poter inserire una sola pagina condivisa con tutti i clienti, sempre nell'area clienti.

Come funziona il plugin?
Una volta attivato, di default è impostato per inserire i file. Potete attivare in backend con il flag anche le altre sezioni direttamente in PRIVATE FILE FOR WOOCOMMERCE/IMPOSTAZIONI.
Una volta attivato, in frontend invece verranno inserite due sezioni in my account di woocommerce nell'area riservata del cliente
SEZ.CONDIVISA=una pagina condivisa con tutti i clienti
SEZ.PRIVATA=infiniti file privati o pagine private al singolo cliente

Le sezioni possono essere attivate/disattivate singolarmente, in base alle proprie esigenze

Il plugin funziona su tutti i temi, non avrai problemi di incompatibilità o conflitti.

Protezione perfetta dei file, nessun direct link per il download neanche se l'utente è loggato e verrebbe a conoscenza del link del file di un altro utente.

Plugin pratico, intuitivo e leggero.

Spero che soddisfi le vostre esigenze, buon lavoro a tutti.


Note:

- Il plugin richiede Woocommerce.
- La dimensione dei file singoli da allegare è in base al vostro hosting. (media max 20mb)
- Il plugin non permette la cancellazione del file caricato dal pannello, ma cancellabile solo in ftp



== Installazione ==

Puoi caricare il file zip da Plugin/Aggiungi Nuovo, dal pannello di amministrazione del tuo wordpress 
oppure
Estrai il file zip e rilascia il contenuto nella directory wp-content / plugins / della tua installazione di WordPress, quindi attiva la pagina Plugin da Plugins.
